RouterControl is the main class, RController is the only other class used.
When first run, Create Routers and Display Routers will display automatically.
You will see 4 rows: each row is information of routers and what they hold.

Program will prompt you to enter your source and destination number.
For this, please refer to the topology demanded in project outline by professor.
As a check, the program will return source and destination client.

Finally, in loops timed for 3 minutes, the program will return the following:
Seeking source and destination client:
Finding Paths...
Choosing Best Path...
Destination: X, Path = [Y], Bandwidth = Z, Cost = C
Where X, Y, Z, and C represent their respective varaibles according to the choices made.
